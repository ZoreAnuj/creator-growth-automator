# Creator Growth Automator

A high-performance automation engine and observability dashboard designed to scale content creation and marketing operations. It provides a centralized control plane to manage, execute, and monitor automated workflows, enabling data-driven growth strategies.

## Key Features
*   **Workflow Automation Engine:** Orchestrates and executes complex, multi-step creator marketing campaigns.
*   **Real-time Observability Dashboard:** Monitors campaign performance, system health, and key metrics.
*   **Production-Ready Scalability:** Built to handle high-volume operations with reliability.
*   **Centralized Campaign Management:** Unified interface to launch and control automated processes.

## Tech Stack
*   Backend: Node.js, Express
*   Database: PostgreSQL
*   Dashboard: React, Next.js
*   Infrastructure: Docker, AWS

## Getting Started
1.  Clone the repository: `git clone https://github.com/zoreanuj/creator-growth-automator.git`
2.  Install dependencies: `npm install`
3.  Set up environment variables in a `.env` file.
4.  Run the development server: `npm run dev`
